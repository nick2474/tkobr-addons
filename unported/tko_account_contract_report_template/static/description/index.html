Contract Template

Define a contract template and send it filled and signed by email

This module adds a contract template tab in "Sales --> Contract", where you can set the contract to use.

After you send by email, it will remains in Sent state untill reception signed by client. After that set it in Progress.

Contract Wizard:

Contract wizard showing most used fields before print.

Any errors in contract template will add red errors in report, easy to see and correct.

Contract Template with Keywords:

Write your contract template in a HTML editor, for better text formating, and set the signature.

You can even add a style CSS to set the document wide formating styles.

Use keywords from all Odoo database, any fields that you can access from this contract are available.

For example the customer name $(partner_id.name)s.
Set images width an height (or just one of it) in keyword $(contract_template_body_id.signature,150,150)s, this will print the signature with 150x150. If width or height is blank or zero it will use image size.
